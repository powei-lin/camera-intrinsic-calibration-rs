# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability, please report it responsibly by emailing powei.lin@proton.me instead of creating a public issue.

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and provide a more detailed response within 7 days indicating our next steps.

## Security Updates

Security updates will be released as soon as possible after verification. Users are encouraged to update promptly.
